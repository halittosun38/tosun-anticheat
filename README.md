# Tosun Anticheat

**[Tosun Anticheat](https://tosundev.com)** — FiveM sunucuları için geliştirilmiş, gerçek zamanlı hile tespiti yapan gelişmiş anticheat ve web panel çözümü.

**[Tosun Anticheat](https://tosundev.com)** — Advanced anticheat & web panel solution for FiveM servers with real-time cheat detection.

## Özellikler / Features

🛡️ Gerçek zamanlı hile tespiti (godmode, freecam, teleport, weapon spawn ve daha fazlası)  
🔍 İmza taraması ile bilinen cheat modülleri tespiti  
🌍 9 dil desteği (TR, EN, DE, ES, FR, RU ve daha fazlası)  
📊 Web tabanlı SaaS yönetim paneli  
🔌 SQL'siz bridge modu desteği
